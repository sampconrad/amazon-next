# Amazon-next

## About

Amazon clone built with NEXT.Js, Redux, Firebase, Auth.Js & Tailwind CSS.

![preview](./preview.png)

# Installation Steps

## Using npm

Run commands

1. `npm install`

2. `npm run dev`

## Or using yarn

Run commands

1. `npm install --global yarn`

2. `yarn install`

3. `yarn run dev`
